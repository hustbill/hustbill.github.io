func multiplyPriors(n: Int) -> Int {
  switch n {
  case 0:
    return 1
  case 1:
    return 2
  default:
    return multiplyPriors(n-1) * multiplyPriors(n-2)
  }
}

算法复杂度是____
选项是 O(n), O(2n), O(n^2), O(nlogn)
