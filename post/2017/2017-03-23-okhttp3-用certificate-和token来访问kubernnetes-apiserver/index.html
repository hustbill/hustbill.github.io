okhttp3 用certificate 和token来访问kubernnetes APIServer

token 位于 /etc/kubernetes/pki/路径下， token.csv

可以通过curl command来访问 curl -k -v GET -h “Authorization : Bearer ” https://apiserver-ip:6443

也可以通过java代码来访问。 client = new httpClient () .header(“Authorization”, StringCpy(" Bearer :%s", token) .builder()
